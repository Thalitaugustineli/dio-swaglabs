# Projeto SWAG LABS

Projeto proposto para fins educacionais, simulando um ambiente de e-commerce para testes

## Descrição
O SWAG LABS é uma aplicação web que simula uma loja online, permitindo aos usuários navegar por produtos, adicionar itens ao carrinho e finalizar compras. Este projeto é ideal para praticar testes automatizados, desenvolvimento front-end e back-end.

## Funcionalidades
- Navegação por categorias de produtos
- Adição e remoção de itens do carrinho
- Processo de checkout
- Autenticação de usuários
- Interface responsiva
